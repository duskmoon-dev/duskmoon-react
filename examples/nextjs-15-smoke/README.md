# Next.js 15 Smoke Test

This is a smoke test example for `@duskmoon-dev/components` using Next.js 15.

## Usage

First, ensure the components package is built from the root:

```bash
bun run build:all
```

Then, run the development server:

```bash
bun run dev
```

Or build the example:

```bash
bun run build
```
